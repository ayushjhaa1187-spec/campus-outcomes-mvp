# Campus Outcomes MVP

> Campus Placement & NIRF Analytics Platform — MVP for college placement data management and NIRF ranking automation.

![build](https://img.shields.io/badge/build-passing-brightgreen) ![license](https://img.shields.io/badge/license-MIT-blue) ![stack](https://img.shields.io/badge/stack-HTML%20%7C%20Node.js-orange)

## 🌐 Live Demo
**[Launch Campus Outcomes →](https://campus-outcomes-mvp.vercel.app)**

## 📖 The Story
> *"The placement cell was drowning in spreadsheets. NIRF rankings were a guessing game. Campus Outcomes gave them a single dashboard — placement trends, recruiter analytics, and NIRF parameter tracking. The data told the story the college needed to hear."*

## ✨ Features
| Feature | Description |
| :--- | :--- |
| **Placement Analytics** | Track placement rates, salary distributions, and recruiter trends |
| **NIRF Dashboard** | Automated calculation of NIRF parameters with visual scoring |
| **Data Management** | Bulk upload and management of student placement records |
| **Report Generation** | Export placement reports in PDF/Excel format |
| **Year-over-Year Comparison** | Track institutional performance across academic years |

## 🛠 Tech Stack
| Layer | Technology |
| :--- | :--- |
| Frontend | HTML5, CSS3, JavaScript |
| Backend | Node.js, Express |

## 🚀 Quick Start
```bash

cd campus-outcomes-mvp
npm install
npm run dev
```
